# orjson

### 參考
+ 🔗 [**GitHub**](https://github.com/ijl/orjson)
+ 🔗 [**myapollo**](https://myapollo.com.tw/blog/python-orjson/)

### 使用原因
1. 它支援 `dataclass`, `datetime`, `enum`, `numpy`, `uuid` (標準庫 json 不支援這些類型的序列化)
2. 它超快 (是 Python JSON library 中序列化速度最快的)
